Audio Transcriber & Project Generator

This application allows you to generate full project assets (documentation, wireframes, and an interactive prototype) and deploy them directly to GitHub Pages. You can trigger the generation process by either uploading an audio/video file for transcription or by providing a text description.

After a successful generation and deployment, you will receive direct links to your new GitHub repository and the live GitHub Pages site.

Upload an audio or video file here. The audio will be transcribed, and the resulting text will be used to generate your project assets. Your project will then be automatically deployed to a new GitHub repository with GitHub Pages enabled.
