Wireframe Guidelines

📱 Mobile-First Approach

All wireframes start with mobile layout (360-428px) and progressively enhance for larger screens.

🔗 Clickable Navigation

Interactive elements allow you to experience user flows and understand page relationships.

📝 Annotation Notes

Yellow callouts explain layout decisions, interaction patterns, and responsive behavior.

⚡ Performance Focus

Layouts prioritize fast loading, accessibility, and smooth user experience across devices.
